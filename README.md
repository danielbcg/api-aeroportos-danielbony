# API REST: Gerenciamento de Aeroportos

## Descrição
API REST completa para gerenciamento de aeroportos utilizando dados do OpenFlights.

## Tecnologias Utilizadas
- Java 17
- Spring Boot 3.1.5
- Spring Data JPA
- H2 Database (testes)
- MySQL (produção)
- Maven
- JUnit 5
- Mockito

## Configuração do Ambiente

### Pré-requisitos
- Java 17 ou superior
- Maven 3.6+
- Git
- MySQL (opcional para desenvolvimento)
